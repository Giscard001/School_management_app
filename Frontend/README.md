# Frontend - Système de Gestion Scolaire

Application Vue.js pour le système de gestion scolaire.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur http://localhost:3000

## Build

```bash
npm run build
```

## Structure

- `/src/views` - Vues par type d'utilisateur
- `/src/layouts` - Layouts pour chaque type d'utilisateur
- `/src/router` - Configuration des routes

## Types d'utilisateurs

- Admin
- Élève
- Professeur
- Parent
- Censeur
- CPE (Conseiller d'Éducation)
- Proviseur
- Secrétaire
